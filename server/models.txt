The required entities:

1. User
    * userId    (String)
    * username  (String)
    * password  (String)
    * firstName (String)
    * lastName  (String)
    * email     (String)
    * role      (Number)

2. Restaurant
    * restaurantId              (String)
    * Name                      (String)
    * businessEmail             (String Email)
    * address                   (String)
    * postalCode                (String)
    * contactInfo               (Number)
    * about                     (String)
    * rating                    (Number)
    * menu                      (String)
    * logo                      (String Img)
    * seatingArrangements[] {   (JSON Array)
        *tableNumber            (Number)
        *tableCapacity          (Number)
    }

3. Menu
    * menuId        (String)
    * Restaurant    --ref (Obj Ref)
    * menuName      (String)
    * category      (String)
    * menuItems[] { (JSON Array)
        *itemName   (String)
        *itemImage  (String Img)
        *about      (String)
        *price      (Number)
    }

4. Reservation
    * reservationId (String)
    * Restaurant    --ref (Obj Ref)
    * User          --ref (Obj Ref)
    * date          (date)
    * slotInterval  (datetime)
    * paymentInfo   (String)
    * tableNumber   (Number)
    * status        (String)
